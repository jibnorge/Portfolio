Leveraging AI to Understand Airbnb Rental Price Dynamics

A comprehensive machine learning initiative designed to develop an accurate predictive model for rental prices in New York City. Leveraging data from Inside Airbnb, which includes 48,895 listings with various features such as location, property type, and amenities, the project aims to help hosts set competitive prices and provide renters with reliable cost estimates for their potential stays.The methodology involved extensive data preparation, exploratory data analysis, and feature engineering. After evaluating nine different regression models, XGBoost emerged as the top performer with an R-squared score of 0.64, the lowest Mean Squared Error (MSE), and the ability to explain 64% of the variance in rental pricing. This robust model demonstrates the project's success in creating a valuable tool for understanding and predicting Airbnb pricing dynamics in the complex New York City market.

AWS Data Engineering Pipeline: From S3 to QuickSight

A comprehensive AWS Data Engineering Capstone Project showcases a robust ETL workflow leveraging various AWS services, demonstrating the seamless integration of Amazon S3, AWS Cloud9, AWS Glue, Amazon Athena, and Amazon QuickSight. The pipeline begins with data extraction from S3, transforms it to Apache Parquet format using Pandas in Cloud9 IDE, and loads it back into S3 for querying with S3 Select. AWS Glue and its crawler create tables and extract metadata, while Athena is used for querying and transforming data into views. Finally, QuickSight visualizes the data, illustrating a real-world scenario of data processing from raw storage to actionable insights within the AWS ecosystem.
